# 数据与授权说明

## 1. 代码和数据是两件事

本仓库开源的是：

- 代码
- 配置样例
- 文档
- 测试和工作台

本仓库不开源也不提供：

- Tushare 账号权限
- 真实 Token
- 真实下载后的全量数据库
- 任何第三方受限授权数据

## 2. 数据获取方式

使用者必须自行：

1. 申请并持有自己的 Tushare 使用权限
2. 配置自己的 `TUSHARE_TOKEN`
3. 在本地执行数据同步脚本
4. 在本地生成自己的 SQLite 数据库

## 3. 不应提交到仓库的文件

下面这些文件默认不应该提交：

- `data_cache.db`
- `*.db-wal`
- `*.db-shm`
- `outputs/`
- `tmp/`
- 本地调试日志

## 4. 口径说明

本项目已经尽量做了“数据全量校验”，但请注意：

- 部分 warning 来自源端历史覆盖限制
- 不同接口的最早日期未必完全一致
- 源端未来更新后，校验结果可能变化

因此，本项目提供的是“当前校验口径下的可验证状态”，不是对第三方源数据永久不变的承诺。

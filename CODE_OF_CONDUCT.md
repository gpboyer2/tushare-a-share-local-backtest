# 社区行为准则

为了让这个项目保持高质量、可合作、可持续，所有参与者默认遵守以下规则。

## 1. 基本原则

- 使用事实沟通，不做人身攻击
- 讨论代码、逻辑、数据和验证，不做情绪化争论
- 对问题保持严格，对人保持克制
- 不用羞辱、嘲讽、阴阳怪气的方式交流

## 2. 我们欢迎什么

- 清晰的问题描述
- 可复现的缺陷报告
- 有验证证据的改进建议
- 结构化的代码评审意见
- 对文档、测试、工程治理的补充

## 3. 我们不接受什么

- 人身攻击
- 歧视性表达
- 恶意挑衅
- 明知存在风险却诱导提交敏感信息
- 重复刷屏、恶意灌水、无意义争论

## 4. 技术讨论的最低标准

提出观点时，尽量同时给出：

- 影响范围
- 具体例子
- 预期行为
- 当前行为
- 验证方法

只说“这样不好”“感觉不对”通常不够。

## 5. 维护者处理原则

维护者有权对以下内容进行限制、隐藏、关闭或拒绝：

- 明显不符合本项目范围的内容
- 明显缺乏基本技术事实支撑的争论
- 含敏感信息的公开内容
- 破坏协作秩序的行为

## 6. 沟通语言

本项目默认使用中文沟通和中文文档。
